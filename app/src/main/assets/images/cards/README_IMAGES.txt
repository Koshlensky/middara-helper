Папка для изображений карт Middara
===================================

Разместите PNG-файлы карт в этой папке.
Каждый файл должен соответствовать полю "imageAsset" в файле data/cards.json.

Примеры:
  images/cards/hero_rook.png
  images/cards/weapon_iron_sword.png
  images/cards/armor_leather.png
  ...

Рекомендуемый размер: 450 x 630 пикселей (соотношение сторон 5:7)
Формат: PNG (поддерживается прозрачность)

Если изображение не найдено, приложение отображает заглушку с цветом типа карты и её названием.
